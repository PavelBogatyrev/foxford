Функция - значение римского числа
Дано число, записанное римскими цифрами. Определите его значение.

Решение оформите в виде функции

int roman2int(char * s)
Функция получает на вход строку, содержащую запись числа римскими цифрами и заканчивающуюся символом с кодом 0. Функция возвращает значение этого числа.

Строка может содержать только символы I, V, X, L, C, D, M, имеющие следующие значения:

Цифра	Значение
I	1
V	5
X	10
L	50
C	100
D	500
M	1000
Кроме этого, следующие последовательности из двух символов имеют специальные значения:

Последовательность	Значение
IV	4
IX	9
XL	40
XC	90
CD	400
CM	900
Гарантируется, что переданная строка - корректная, то есть является записью некоторого числа в римской системе счисления. Число может принимать значения от 1 до 3999.

На проверку вам нужно сдать файл, в котором будет только тело этой функции, и не будет функции 
main
. При проверке этой задачи к вашей программе будет добавлена следующая функция 
main
:

int main()
{
    char s[20];
    scanf("%s", s);
    printf("%d\n", roman2int(s));
    return 0;
}
после чего решение будет проверено на тестах.
